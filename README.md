## ❓ Python là gì và tại sao chọn Python?

Python là ngôn ngữ lập trình bậc cao, dễ học, dễ đọc, có hệ sinh thái thư viện khổng lồ. Nó được ưa chuộng vì:

- **Đa năng**: từ web, khoa học dữ liệu, AI, đến tự động hóa.
- **Cộng đồng lớn**: hỗ trợ mạnh mẽ, nhiều tài nguyên.
- **Tích hợp dễ dàng**: với C/C++, Java, .NET, v.v.

---

## ⚠️ Các lưu ý khi làm việc với Python

- **Quản lý phiên bản**: Sử dụng `pyenv` hoặc `virtualenv` để tránh xung đột giữa các dự án.
- **PEP 8**: Tuân thủ quy tắc định dạng mã nguồn để code dễ đọc.
- **Kiểu dữ liệu động**: Cần chú ý khi thao tác với biến để tránh lỗi không mong muốn.
- **Xử lý ngoại lệ**: Luôn dùng `try/except` để bắt lỗi khi nhập/xuất dữ liệu.

## 🛠️ Các kỹ năng cần biết

Dưới đây là những kỹ năng cơ bản bạn cần có để bắt đầu:

### 🔹 Để chạy được file Python, bạn cần:

1. **Cài đặt Python**  
   - Tải từ [python.org](https://python.org) và thêm vào PATH.
2. **Cài đặt IDE**  
   - Gợi ý: VS Code, PyCharm, Jupyter Notebook.
3. **Cài đặt Extension** (nếu dùng VS Code)  
   - Python extension của Microsoft.
4. **Cài đặt Package**  
   - Sử dụng `pip install <tên-package>` (ví dụ: `pip install numpy`).

## 🚀 Hướng dẫn chạy file Python

### Trên Windows
Mở Command Prompt hoặc PowerShell và gõ:
```bash
python ten_file.py
```
### Trên Linux / macOS
Mở Terminal và gõ:
```bash
python3 ten_file.py
```
